# 💻 Simulador de Ahorro para PC Gamer

¡Bienvenido! Este es mi primer proyecto formal en Python. Es un asistente interactivo diseñado para ayudar a cualquier gamer a planificar su presupuesto de hardware de forma inteligente.

## 🚀 Características Principales
* **🛡️ Escudo Anti-Errores:** Implementación de bloques `try/except` para evitar que el programa falle si el usuario ingresa datos inválidos.
* **🌎 Multimoneda:** Soporte dinámico para cualquier moneda local (soles, dólares, pesos, etc.).
* **🎟️ Sistema de Cupones:** Lógica condicional para aplicar descuentos exclusivos (ej. Cupón GIUX20).
* **📂 Historial de Consultas:** Registro automático de cada simulación en un archivo `.txt` con marca de tiempo real.
* **🧮 Cálculos Automáticos:** Uso de la librería `math` para determinar semanas exactas de ahorro.

## 🛠️ Tecnologías Usadas
* **Python 3.x**
* **Librerías:** `datetime` (gestión de fechas), `math` (cálculos matemáticos pro).

## 📖 Cómo usarlo
1. Clona este repositorio o descarga el archivo `pcgamer.py`.
2. Ejecuta el script en tu terminal con `python pcgamer.py`.
3. Sigue las instrucciones en pantalla para configurar tu presupuesto.
